# Hospital-Emergency-Room-Dashboard-Excel
This project is an interactive Hospital Emergency Room Dashboard created in Microsoft Excel. The dashboard helps visualize hospital patient data and provides meaningful insights using charts, KPIs, slicers, and data visualization techniques.

**Features**
Interactive Dashboard
KPI Cards
Pivot Tables & Pivot Charts
Dynamic Charts
Slicers & Filters
Patient Analysis
Department Referral Analysis
Real-time Date & Time
Professional Dashboard Design

**Dashboard Insights**
Number of Patients
Average Wait Time
Patient Satisfaction Score
Admission Status
Gender Analysis
Age Group Analysis
Department Referral Analysis

**Tools Used**
Microsoft Excel, Pivot Tables, Pivot Charts, Conditional Formatting, Data Visualization, DAX Formulas
